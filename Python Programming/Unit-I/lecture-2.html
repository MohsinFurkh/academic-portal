Lecture 2: Python Data Types and Tokens

1. Dynamic Typing

Python is dynamically typed, meaning variable types are determined at runtime.

x = 5 # x is an integer
x = "Hello" # Now x is a string

2. Mutable and Immutable Data Types

Mutable Types Immutable Types
Lists Numbers (int, float, complex)
Dictionaries Strings
Sets Tuples
Byte Arrays Frozen Sets

3. Python Tokens

Python has several types of tokens:

  • Keywords: Reserved words with special meaning (e.g., if, else, while)
  • Identifiers: Names given to variables, functions, classes, etc.
  • Literals: Fixed values (numbers, strings, etc.)
  • Operators: Symbols that perform operations (+, -, *, /, etc.)
  • Punctuators: Special symbols like [], {}, (), :, etc.

4. Naming Conventions

  • Variable and function names: snake_case (e.g., student_name, calculate_average)
  • Class names: PascalCase (e.g., StudentRecord, BankAccount)
  • Constants: UPPER_CASE (e.g., MAX_VALUE, PI)
  • Private variables: _single_leading_underscore
  • Name mangling: __double_leading_underscore

5. type() and id() Functions

type() Function

Returns the type of an object.

x = 5
print(type(x)) # <class 'int'>

y = "Hello"
print(type(y)) # <class 'str'>

id() Function

Returns the identity (memory address) of an object.

a = [1, 2, 3]
b = a
print(id(a)) # Memory address of a
print(id(b)) # Same as id(a) because they reference the same object
