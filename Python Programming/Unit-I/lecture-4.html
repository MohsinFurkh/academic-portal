Lecture 4: Operators and Decision Making

1. Operators in Python

Arithmetic Operators

Operator Description Example
+ Addition 5 + 3 → 8
- Subtraction 5 - 3 → 2
* Multiplication 5 * 3 → 15
/ Division (float) 5 / 2 → 2.5
// Floor division 5 // 2 → 2
% Modulus 5 % 2 → 1
** Exponent 2 ** 3 → 8

Relational/Comparison Operators

a, b = 5, 3
print(a > b) # True
print(a == b) # False
print(a != b) # True
print(a <= b) # False

Logical Operators

x = True
y = False
print(x and y) # False
print(x or y) # True
print(not x) # False

Assignment Operators

x = 5
x += 3 # x = x + 3 → 8
x -= 2 # x = x - 2 → 6
x *= 2 # x = x * 2 → 12

Bitwise Operators

a = 5 # 0101 in binary
b = 3 # 0011 in binary
print(a & b) # AND → 1 (0001)
print(a | b) # OR → 7 (0111)
print(a ^ b) # XOR → 6 (0110)
print(~a) # NOT → -6 (2's complement)
print(a << 1) # Left shift → 10 (1010)
print(a >> 1) # Right shift → 2 (0010)

Membership Operators

numbers = [1, 2, 3, 4, 5]
print(3 in numbers) # True
print(6 not in numbers) # True

Identity Operators

x = [1, 2, 3]
y = [1, 2, 3]
z = x
print(x is y) # False (different objects)
print(x is z) # True (same object)
print(x == y) # True (same value)

2. Operator Precedence and Associativity

Operator Precedence (Highest to Lowest)

  1. ** (Exponentiation)
  2. ~ + - (Bitwise NOT, Unary plus and minus)
  3. * / % // (Multiplication, Division, Modulus, Floor division)
  4. + - (Addition, Subtraction)
  5. >> << (Bitwise shift operators)
  6. & (Bitwise AND)
  7. ^ | (Bitwise XOR, OR)
  8. <= < > >= (Comparisons)
  9. == != (Equality operators)
  10. = %= /= //= -= += *= **= (Assignment operators)
  11. is, is not, in, not in (Identity, membership operators)
  12. not or and (Logical operators)
Note: Use parentheses () to change the order of evaluation.

3. Decision Making Statements

if Statement

age = 20
if age >= 18:
    print("You are an adult")

if-else Statement

age = 15
if age >= 18:
    print("You can vote") else:
    print("You cannot vote")

elif Statement

score = 85
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
else:
    grade = 'F'
print(f"Your grade is {grade}")

Nested if Statement

age = 20
has_id = True

if age >= 18:
    if has_id:
        print("You can enter the club")     else:
        print("You need an ID to enter") else:
    print("You are too young to enter")

match Statement (Python 3.10+)

def http_status(status):
    match status:
        case 200:
            return "OK"
        case 404:
            return "Not found"
        case 500 | 502 | 503:
            return "Server error"
        case _:
            return "Unknown status code"
