Lecture 3: Strings and I/O

1. String Values

Strings in Python can be created using single, double, or triple quotes.

str1 = 'Single quoted string'
str2 = "Double quoted string"
str3 = '''Multi-line
string'''
str4 = """Another multi-line
string"""

2. String Methods

Python provides many built-in string methods:

Method Description Example
upper() Converts to uppercase 'hello'.upper() → 'HELLO'
lower() Converts to lowercase 'HELLO'.lower() → 'hello'
strip() Removes whitespace ' hello '.strip() → 'hello'
split() Splits string into list 'a,b,c'.split(',') → ['a','b','c']
replace() Replaces substring 'hello'.replace('h','j') → 'jello'

3. String Operators

Operator Description Example
+ Concatenation 'Hello' + 'World' → 'HelloWorld'
* Repetition 'Hi' * 3 → 'HiHiHi'
[] Indexing 'Hello'[1] → 'e'
in Membership 'e' in 'Hello' → True

4. format() Method

The format() method formats the specified value(s) and inserts them inside the string's placeholder.

# Basic formatting
print("Hello, {}. You are {} years old.".format("Alice", 25))
# Output: Hello, Alice. You are 25 years old.

# Positional arguments
print("{1} {0} {2}".format("be", "to", "or not to"))
# Output: to be or not to

# Named arguments print("Name: {name}, Age: {age}".format(name="Bob", age=30))

5. Escape Sequence Characters

Escape Sequence Description
\n New line
\t Tab
\\ Backslash
\' or \" Single or double quote
print("Line 1\nLine 2\tIndented")

6. Input and Output Functions

Input Function

name = input("Enter your name: ") print("Hello, " + name)

Output Formatting (f-strings in Python 3.6+)

name = "Alice" age = 25 print(f"{name} is {age} years old.")

Multiple Values

x, y = 10, 20 print(x, y, sep=" | ", end="!\n")
