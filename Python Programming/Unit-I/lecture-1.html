Lecture 1: Introduction to Python

1. Working with Python

Python is a high-level, interpreted programming language known for its simplicity and readability.

Key Features:
  • Easy to learn and use
  • Dynamically typed
  • Supports multiple programming paradigms
  • Large standard library

2. Interactive Mode and Scripting Mode

Interactive Mode

>> print("Hello, World!")
Hello, World!

Scripting Mode

Create a file with .py extension and run it using Python interpreter.

# hello.py
print("Hello, World!")

Run with: python hello.py

3. Basic Syntax

# This is a comment
print("Hello, World!") # This prints a message

Python uses indentation to define code blocks instead of curly braces.

4. Comments

# This is a single-line comment

""" This is a multi-line comment (docstring) """

5. Indentation

Python uses indentation to indicate blocks of code.

if 5 > 2:
    print("Five is greater than two!") # This is indented
Note: Use 4 spaces per indentation level (PEP 8 recommendation).
