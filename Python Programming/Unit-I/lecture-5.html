Lecture 5: Loops in Python

1. range() Function

The range() function generates a sequence of numbers, commonly used for looping a specific number of times.

Basic Syntax

# range(stop) for i in range(5): print(i, end=' ') # Output: 0 1 2 3 4

With Start and Stop

# range(start, stop) for i in range(2, 6): print(i, end=' ') # Output: 2 3 4 5

With Step

# range(start, stop, step) for i in range(0, 10, 2): print(i, end=' ') # Output: 0 2 4 6 8

Negative Step

# Counting down for i in range(5, 0, -1): print(i, end=' ') # Output: 5 4 3 2 1

2. for Loop

The for loop is used for iterating over a sequence (list, tuple, string, etc.) or other iterable objects.

Iterating over a List

fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(f"I like {fruit}")
I like apple
I like banana
I like cherry

Using enumerate()

fruits = ["apple", "banana", "cherry"] for index, fruit in enumerate(fruits, start=1): print(f"{index}. {fruit}")

3. while Loop

The while loop executes as long as a condition is True.

# Basic while loop count = 1 while count <= 5: print(f"Count: {count}") count += 1

Infinite Loop with break

while True: user_input = input("Enter 'quit' to exit: ") if user_input.lower() == 'quit': break print(f"You entered: {user_input}")

4. Loop Control Statements

break Statement

for i in range(10): if i == 5: break # Exit the loop when i is 5 print(i, end=' ') # Output: 0 1 2 3 4

continue Statement

for i in range(5): if i == 2: continue # Skip the rest of the code for i=2 print(i, end=' ') # Output: 0 1 3 4

pass Statement

for i in range(5): if i % 2 == 0: pass # Do nothing, just a placeholder else: print(f"{i} is odd")

5. else in Loops

The else block after a loop executes only if the loop completes normally (not exited by break).

# Else with for loop for i in range(3): print(i) else: print("Loop completed!")
0
1
2
Loop completed!
# Else not executed with break for i in range(3): if i == 1: break print(i) else: print("This won't print")

6. Nested Loops

A loop inside another loop is called a nested loop.

Nested for Loop

# Multiplication table for i in range(1, 4): for j in range(1, 4): print(f"{i} * {j} = {i * j}") print("--------")

Pattern Printing

# Right triangle pattern n = 5 for i in range(1, n + 1): for j in range(i): print("*", end="") print()

7. List Comprehensions

A concise way to create lists using loops in a single line.

# Traditional way squares = [] for x in range(10): squares.append(x ** 2) # Using list comprehension squares = [x ** 2 for x in range(10)] # With condition even_squares = [x ** 2 for x in range(10) if x % 2 == 0]
