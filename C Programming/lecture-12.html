Introduction to Strings

In C programming, a string is a sequence of characters terminated with a null character '\0'.

Key Characteristics:

  • Stored as arrays of characters
  • End with a null terminator '\0'
  • Can be initialized using string literals or character arrays
  • Support various operations through standard library functions

String Declaration and Initialization

                    // Different ways to declare and initialize strings
                    char str1[] = "Hello";  // Automatic size calculation
                    char str2[10] = "World"; // Fixed size with initialization
                    char str3[] = {'H', 'e', 'l', 'l', 'o', '\0'}; // Character array
                    char str4[10]; // Declaration without initialization

String Input/Output

Reading and Writing Strings

                    #include <stdio.h>
                    
                    int main() {
                        char name[50];
                        
                        // Reading a string
                        printf("Enter your name: ");
                        scanf("%s", name);  // Reads until whitespace
                        
                        // Reading a line of text
                        printf("Enter your full name: ");
                        fflush(stdin);  // Clear input buffer
                        fgets(name, 50, stdin);  // Reads entire line
                        
                        // Writing strings
                        printf("Hello, %s\n", name);
                        puts(name);  // Alternative to printf
                        
                        return 0;
                    }
Important: Always ensure the input doesn't exceed the array size to prevent buffer overflow. Consider using fgets() instead of gets() as it's safer.

String Handling Functions

The string.h library provides several useful functions for string manipulation.

Common String Functions

                    #include <string.h>
                    
                    int main() {
                        char str1[20] = "Hello";
                        char str2[20] = "World";
                        char str3[20];
                        int len, cmp;
                        
                        // String length
                        len = strlen(str1);  // Returns 5
                        
                        // String copy
                        strcpy(str3, str1);  // str3 = "Hello"
                        
                        // String concatenation
                        strcat(str1, str2);  // str1 = "HelloWorld"
                        
                        // String comparison
                        cmp = strcmp(str1, str2);  // Returns 0 if equal
                        
                        return 0;
                    }

String Function Cheat Sheet:

  • strlen() - Get string length
  • strcpy() - Copy one string to another
  • strcat() - Concatenate two strings
  • strcmp() - Compare two strings
  • strstr() - Find substring in string
  • strchr() - Find character in string

String Manipulation

Common String Operations

                    #include <string.h>
                    #include <ctype.h>
                    
                    int main() {
                        char text[100] = "  C Programming is Fun!  ";
                        char *token;
                        
                        // Converting case
                        for(int i = 0; text[i]; i++) {
                            text[i] = toupper(text[i]);
                        }
                        
                        // String tokenization
                        token = strtok(text, " ");
                        while(token != NULL) {
                            printf("%s\n", token);
                            token = strtok(NULL, " ");
                        }
                        
                        // String reversal
                        strrev(text);  // Non-standard but common in some compilers
                        
                        return 0;
                    }
Note: The strrev() function is not part of the standard C library but is commonly provided by many compilers. You may need to implement your own string reversal function for maximum portability.

Best Practices and Common Pitfalls

Common Mistakes to Avoid

  • Buffer Overflow: Always check string lengths before operations
  • Missing Null Terminator: Ensure strings are properly null-terminated
  • Unsafe Functions: Avoid gets(), prefer fgets()
  • Memory Management: Be careful with dynamic string allocation

Safe String Handling

                    // Safe string copy with bounds checking
                    #define MAX_LEN 50
                    
                    void safeStringCopy(char *dest, const char *src, size_t dest_size) {
                        if (dest_size == 0) return;
                        strncpy(dest, src, dest_size - 1);
                        dest[dest_size - 1] = '\0';
                    }
                    
                    // Safe string concatenation
                    void safeStringCat(char *dest, const char *src, size_t dest_size) {
                        size_t dest_len = strlen(dest);
                        if (dest_len >= dest_size) return;
                        strncat(dest, src, dest_size - dest_len - 1);
                    }
Security Tip: Always validate and sanitize string inputs, especially when they come from external sources, to prevent security vulnerabilities like buffer overflows and injection attacks.
